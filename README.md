Job Portal Web Application

A modern, interactive Job Portal Web Application built using HTML, CSS, and JavaScript. This project demonstrates dynamic UI rendering, real-time filtering, and job bookmarking without page reload, making it suitable for academic micro projects and beginner web development practice.

---

 Live Features

Real-time Job Search
 Filter by Company
 Filter by Job Role
Save / Bookmark Jobs
Responsive Design (Mobile-Friendly)
Dynamic UI Updates (No Reload Required)
Modern Card Layout with Hover Effects
 Tech Stack

Technology| Description
HTML5| Structure of the web application
CSS3| Styling (Flexbox + Grid layout)
JavaScript (ES6)| Dynamic functionality
TypeScript Concepts| Structured coding approach

Functional Overview

- Search Bar → Filters jobs instantly based on title
- Company Dropdown → Displays jobs from selected company
- Role Filter → Filters jobs by role (Developer, Designer, Manager)
- Save Button → Adds job to "Saved Jobs" section dynamically
